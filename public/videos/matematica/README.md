# Vídeos do DAVI Escola — Matemática

Coloque aqui os vídeos **autorais** das aulas (produzidos pelo DAVI), em `.mp4`.
Enquanto o arquivo não existir, a página da aula mostra o placeholder
_"Vídeo demonstrativo autoral do Projeto DAVI em preparação."_ e o restante
(apoio visual, exercícios, teclado numérico, TTS, varredura) segue funcionando.

Cada aula procura o arquivo `/videos/matematica/{id}.mp4`, onde `{id}` é o
identificador da aula em `app/lib/matematica.ts`. Não é preciso mexer no código.

## Nomes esperados

| Aula | Arquivo |
|------|---------|
| Números de 0 a 10 | `numeros-0-10.mp4` |
| Números de 0 a 20 | `numeros-0-20.mp4` |
| Contar objetos | `contar-objetos.mp4` |
| Antes e depois | `antes-depois.mp4` |

| Soma simples | `soma-simples.mp4` |
| Subtração simples | `subtracao-simples.mp4` |
| Multiplicação simples | `multiplicacao-simples.mp4` |
| Divisão simples | `divisao-simples.mp4` |
| Complete a sequência | `sequencia.mp4` |

Formato: MP4 (H.264) ou WebM. Nenhum vídeo de terceiros é usado.
