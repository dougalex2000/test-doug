# Vídeos do DAVI Escola — Português

Coloque aqui os vídeos **autorais** das aulas (produzidos pelo DAVI), em `.mp4`.
Enquanto o arquivo não existir, a página da aula mostra o placeholder
_"Vídeo demonstrativo autoral do Projeto DAVI em preparação."_ e os controles
seguem funcionando com apoio de voz. Para trocar o vídeo de uma aula, basta
colocar o arquivo com o nome esperado — não é preciso mexer no código.

Nenhum vídeo de terceiros (YouTube etc.) é usado.

Formato: MP4 (H.264) ou WebM.

## Roteiro de referência — Português: Família BA BE BI BO BU (~45s)

Arquivo alvo: `portugues-ba-be-bi-bo-bu.mp4`

- **Cena 1 (0s–5s)** — Tela clara. Texto grande: "Vamos aprender as sílabas com B".
  Narração: "Agora vamos aprender algumas sílabas com a letra B."
- **Cena 2 (5s–15s)** — Mostrar uma sílaba por vez: BA, BE, BI, BO, BU.
  Narração: "BA... BE... BI... BO... BU."
- **Cena 3 (15s–25s)** — Mostrar: BA + LA = BALA.
  Narração: "Com BA, podemos formar a palavra BALA."
- **Cena 4 (25s–35s)** — Mostrar: BO + LA = BOLA.
  Narração: "Com BO e LA, formamos BOLA."
- **Cena 5 (35s–45s)** — Mostrar: "Agora é sua vez! Escreva uma palavra com BA, BE, BI, BO ou BU."
  Narração: "Agora é sua vez. Escreva uma palavra com uma dessas sílabas. Você consegue!"
