# Vídeos do DAVI Escola — Português

Coloque aqui os vídeos próprios das aulas (produzidos pelo DAVI). Enquanto o
arquivo não existir, a página da aula mostra um card "Vídeo da aula será
inserido aqui" e os controles seguem funcionando com apoio de voz.

Nomes esperados (referenciados em `app/lib/portugues.ts`):

- `introducao.mp4`
- `vogais.mp4`
- `letra-a.mp4`, `letra-b.mp4`, … `letra-z.mp4`
- `ch.mp4`, `lh.mp4`, `nh.mp4`, `bl.mp4`, `br.mp4`, `nasalados.mp4`,
  `qu.mp4`, `gu.mp4`, `rr.mp4`, `ss.mp4`

Formato: MP4 (H.264) ou WebM. Não usar vídeos de terceiros sem autorização.
