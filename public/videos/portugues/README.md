# Vídeos do DAVI Escola — Português

Vídeos **autorais** das aulas (produzidos pelo DAVI), em `.mp4`. Enquanto um
arquivo não existir, a página da aula mostra o placeholder _"Vídeo demonstrativo
autoral do Projeto DAVI em preparação."_ e os controles seguem funcionando com
apoio de voz. Para trocar/adicionar um vídeo, basta colocar o arquivo com o nome
esperado — não é preciso mexer no código.

Um arquivo `poster` (`.png`) opcional, com o mesmo nome do vídeo + `-poster`, é
mostrado como prévia antes de tocar. Nenhum vídeo de terceiros (YouTube etc.).
