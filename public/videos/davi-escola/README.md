# Vídeos autorais — DAVI Escola

Coloque aqui os vídeos demonstrativos **autorais** do Projeto DAVI (formato `.mp4`).
Enquanto o arquivo não existir, a aula mostra automaticamente um placeholder:
_"Vídeo demonstrativo autoral do Projeto DAVI em preparação."_

Nenhum vídeo externo (YouTube etc.) é usado. Para trocar o vídeo de uma aula,
basta colocar o arquivo com o nome indicado abaixo — não é preciso mexer no código.

## Português — nomes de arquivo esperados

| Aula | Arquivo |
|------|---------|
| Aprendendo a ler sílaba por sílaba | `portugues-silaba-por-silaba.mp4` |
| Aula complementar de alfabetização | `portugues-alfabetizacao.mp4` |
| Introdução (Por que aprendemos a escrever?) | `portugues-introducao.mp4` |
| Vogais | `portugues-vogal-a.mp4` |
| Família da Letra B (BA BE BI BO BU) | `portugues-ba-be-bi-bo-bu.mp4` |
| Demais letras | `portugues-letra-{a..z}.mp4` |
| Sons complexos (CH, LH, NH, …) | `portugues-{ch,lh,nh,…}.mp4` |

## Roteiro de referência — Português: Família BA BE BI BO BU (~45s)

Arquivo alvo: `portugues-ba-be-bi-bo-bu.mp4`

- **Cena 1 (0s–5s)** — Tela clara. Texto grande: "Vamos aprender as sílabas com B".
  Narração: "Agora vamos aprender algumas sílabas com a letra B."
- **Cena 2 (5s–15s)** — Mostrar uma sílaba por vez: BA, BE, BI, BO, BU.
  Narração: "BA... BE... BI... BO... BU."
- **Cena 3 (15s–25s)** — Mostrar: BA + LA = BALA.
  Narração: "Com BA, podemos formar a palavra BALA."
- **Cena 4 (25s–35s)** — Mostrar: BO + LA = BOLA.
  Narração: "Com BO e LA, formamos BOLA."
- **Cena 5 (35s–45s)** — Mostrar: "Agora é sua vez! Escreva uma palavra com BA, BE, BI, BO ou BU."
  Narração: "Agora é sua vez. Escreva uma palavra com uma dessas sílabas. Você consegue!"
